# **App Name**: UniVault Website

## Core Features:

- Hero Section: Display a headline, sub-headline, app store download buttons, and a short video loop of the app's UI.
- Problem/Solution Section: Highlight common student problems (missed deadlines, scattered resources) and present UniVault as the solution.
- Feature Sections: Dedicated pages/sections detailing core features: Academic Record, Easy Learning, Smart Reminders, Campus Events, and Analytics.
- Testimonials: Include student testimonials and partner logos to build trust.
- Institution Section: Explain the backend value proposition for institutions (student engagement, streamlined admin, analytics).
- Download Section: Prominent app store download badges and a clear call to action to download the app.
- AI Chatbot Integration: Integrate an AI chatbot that functions as a smart tool capable of answering student questions about campus resources, deadlines, and academic information.

## Style Guidelines:

- Primary color: Soft light blue (#A0D2EB) to create a clean and trustworthy feeling, reflecting the app's purpose of academic organization and support.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop that ensures readability and reduces visual fatigue.
- Accent color: A vibrant orange (#FFB347) for calls to action and key interactive elements, drawing attention and creating a sense of energy.
- Headline font: 'Poppins', sans-serif, for headlines to give a geometric, precise feel.
- Body font: 'PT Sans', sans-serif, for body text, complementing 'Poppins' with a modern, clean, readable look.
- Use clean, minimalist icons that represent each feature (e.g., a calendar for Smart Reminders, a book for Easy Learning).
- Ensure a mobile-responsive layout, optimized for viewing on various devices.
- Incorporate subtle animations and transitions to enhance user engagement and provide visual feedback, like smooth scrolling or a loading animation when accessing the Academic Record.
